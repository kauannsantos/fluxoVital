# FluxoVital

Sistema web para gestão de doadores e instituições de saúde, com foco em doações de sangue, leite materno e medula óssea. Desenvolvido como projeto interdisciplinar.

## Tecnologias

* **Backend:** Java, Spring Boot, Spring Data JPA
* **Banco de Dados:** MySQL
* **Frontend:** HTML, CSS, Bootstrap

## Funcionalidades

* Cadastro e gestão de doadores
* Cadastro de instituições
* Controle de tipos de doação
* Busca e listagem de doadores

## Arquitetura

Estrutura em camadas:

* Controller
* Service
* Repository
* Model (Entity)

## Observações

Projeto acadêmico, com possibilidade de evolução para autenticação, notificações e métricas.
